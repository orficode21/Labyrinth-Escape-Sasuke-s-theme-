-------------Tutorial Gameplay-----------------
Keypad Left	: <-	Untuk bergerak ke kiri
Keypad Right	: ->	Untuk bergerak ke kanan
Keypad Up	: ^	Untuk bergerak ke atas
Keypad Down	: v	Untuk bergerak ke bawah

*Kode program game Labyrinth escape ini dibuat untuk memenuhi projek akhir mata kuliah Object Oriented Programming
