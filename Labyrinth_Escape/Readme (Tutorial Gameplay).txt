----------------------------------------------!!IMPORTANT!!-------------------------------------------------

Game file name: "Labyrinth_Escape.pde"

This program file runs using the Processing Java application, and you need to download the sound library:
- Open the sketch
- Import library
- Manage library
- Search for "Sound" and download it
- Run the game

Ensure no files are missing when extracting, and if you want to play without music, you can add a comment "//" in the code:

// Displaying background music
// file = new SoundFile(this, "Music.mp3");
// file.play();
// file = new SoundFile(this, "Crow Raven.mp3");
// file.loop();

-------------Gameplay Tutorial-----------------
Keypad Left	: <-	To move left
Keypad Right	: ->	To move right
Keypad Up	: ^	To move up
Keypad Down	: v	To move down

*The Labyrinth Escape game program code is created to fulfill the final project of the Object Oriented Programming course.
